About YOLO Object Detection

This Website demonstrates real-time object detection using the YOLO (You Only Live Once) algorithm. We built this with modern tools to deliver accurate and fast detection in real-world applications.

Tools Used

  • YOLOv8
  • Flask (Backend Framework)
  • HTML, CSS (Frontend)

Key Features

  • Real-time image upload and object detection
  • Annotated output with bounding boxes
  • JSON results for easy integration
  • Lightweight Flask backend for fast deployment
  • User-friendly frontend interface
